# Asylpolitik in der EU

Kurzbeschreibung: Analysen und Zusammenstellungen zur Asyl- und Migrationspolitik in der EU. Die Seite enthält kompakte Pro/Contra‑Analysen nach Ländern, Quellenangaben, sowie Links zu Daten und weiterführenden Dokumenten zur politischen Diskussion.

Inhalte in diesem Verzeichnis: `index.html` (Übersicht, Ressourcen, Fazit), `tabloid.html` (kompaktes Pro/Contra‑Format) und zusätzliche Dokumente oder Grafiken, die Sie hier ablegen können.
